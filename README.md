# 🌸 Task Manager

A simple and elegant task management app built with React.

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🔍 Filter tasks — All / Active / Completed
- 💾 Auto-save with localStorage (data persists after refresh)
- 📱 Fully responsive — works on mobile and desktop

---

## 🛠️ Tech Stack

- **React** — UI components
- **useState** — state management
- **useEffect** — side effects
- **localStorage** — persistent storage
- **CSS** — custom Soft Pastel design

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm run dev
```

---

## 💡 Usage

| Action | How |
|--------|-----|
| Add task | Type and press **Enter** or click ** Add** |
| Complete task | Click the checkbox |
| Delete task | Hover over task → click  |
| Filter tasks | Click **All / not Completed / Completed** |
